# 🎬 Wikipedia Movie Scraper (Scrapy)

Парсер на базе фреймворка **Scrapy**, предназначенный для автоматического сбора подробных данных о фильмах из категории «Фильмы по алфавиту» в Русской Википедии.

## 🛠 Настройка проекта
Вы можете адаптировать работу парсера под свои задачи:

1.  **Лимит страниц (`wiki.py`)**:
    * `pages_limit = 10` — парсинг первых n страниц категории (по умолчанию 10).
    * `pages_limit = 'All'` — полный сбор всей базы фильмов.
2.  **Управление данными (`pipelines.py`)**:
    * В этом файле можно изменить путь к выходному файлу или настроить формат вывода. По умолчанию данные пишутся в `movies.csv`.
3.  **Интенсивность запросов (`settings.py`)**:
    * Настроены параметры `AUTOTHROTTLE` и `DOWNLOAD_DELAY`. Это позволяет имитировать поведение человека и избегать блокировок (HTTP 429) со стороны серверов Wikipedia.

## 📦 Как запустить
Для работы проекта вам понадобится Python 3.

1.  **Установите Scrapy**:
    ```bash
    pip install scrapy
    ```
2.  **Перейдите в корневую директорию проекта** (там, где находится файл `scrapy.cfg`).
3.  **Запустите процесс сбора данных**:
    ```bash
    scrapy crawl wiki
    ```

## 📊 Результат
После завершения работы в корневой папке появится (или обновится) файл `movies.csv`. 
*Пример структуры данных доступен в файле `movies2.csv`.*
