# Wikipedia Movie Scraper (Scrapy)

Проект на базе Scrapy для автоматического сбора данных о фильмах из категории «Фильмы по алфавиту» в Русской Википедии.

## Особенности
* **Умная пагинация**: Переходит по страницам категории, используя динамические ссылки «Следующая страница».
* **Обход блокировок**: Настроены User-Agent и AutoThrottle для защиты от ошибки HTTP 429 и бана IP сайтом википедии.
* **Сохранение данных**: Данные по умолчанию сохраняются в movies.csv.

## Параметры настройки
В файле `wiki.py` можно настроить глубину парсинга:
* `pages_limit = 10` — ограничение на количество страниц (по умолчанию собрано ~2000 фильмов).
* `pages_limit = 'All'` — для полного сбора всех фильмов категории.

## Как запустить
1. Установите зависимости: `pip install scrapy`
2. Перейдите в папку проекта.
3. Запустите парсер командой:
   ```bash
   scrapy crawl wiki
